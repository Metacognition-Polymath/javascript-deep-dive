# 7. 연산자

## 7.1 산술 연산자

- 이항 산술 연산자
- 단항 산술 연산자

```js
// 예제 7-6
var x = "1";
// 문자열을 숫자로 변환한다.
console.log(+x); // 1
```

- 문자열 연결 연산자
  - true : 1, false : 0

```js
1 + true;
1 + false;
```

## 7.3 비교 연산자

- 동등/일치 비교 연산자

  - 동등 비교(==) : 값이 같음
  - 일치 비교(===) : 값과 타입이 같음
  - 동등 비교(==) 연산자는 예측하기 어려운 결과를 만들어 내므로 사용하지 않는 편이 좋다.
  - 특이한 경우 : NaN === NaN : false
  - NaN인지 확인하려면 Number.isNaN 메서드를 이용한다.

- 대소 관계 비교 연산자

## 7.4 삼항 조건 연산자

- if - else 문과 차이점
  - 삼항연산자 표현식은 값 처럼 사용할 수 있지만 if - else문은 값처럼 사용할 수 없다.

## 7.5 논리 연산자

- 단축 평가(9.4절에서 자세히 다룸)

```js

```

## 7.9 지수 연산자 => \*\*

```js
2 ** 3; // 8
Math.pow(2, 3); // 8
```

```js
// 음수를 거듭제곱하려면 ()로 묶어야 한다.
(-5) ** 2; // 25
```

- 지수 연산자는 `이항 연산자 중에서` 우선순위가 가장 높다.
